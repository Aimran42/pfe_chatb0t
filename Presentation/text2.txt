Comment faciliter l’accès aux informations de l’EST FBS ?

Comment fournir des réponses rapides et fiables ?
Comment améliorer l’expérience des étudiants et visiteurs ?
Solution proposée : chatbot basé sur le RAG
Slide 6 : Objectifs Du Projet
Développer une interface web de chatbot
Permettre aux utilisateurs de poser des questions
Interroger les documents PDF/TXT de l’EST FBS
Générer des réponses contextualisées
Afficher les sources et sauvegarder les interactions
Slide 7 : Solution Proposée
Application web servie par FastAPI
Interface chatbot en HTML/CSS/JavaScript
Pipeline RAG basé sur LangChain
Recherche vectorielle avec Pinecone
Génération de réponse avec Groq LLM
Slide 8 : Architecture Générale
Frontend : index.html
Backend API : app/main.py
Service NLP/RAG : app/rag.py
Indexation des documents : scripts/load_vectors.py
Journalisation : app/database.py
Slide 9 : Technologies Utilisées
Frontend : HTML, CSS, JavaScript natif
Backend : Python, FastAPI, Uvicorn
NLP/RAG : LangChain, Gemini Embeddings, Groq
Base vectorielle : Pinecone
Données : SQLAlchemy, PostgreSQL/Supabase
