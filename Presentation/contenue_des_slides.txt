Slide 7 : Solution Proposée
Contenu à afficher dans la slide :
Application web servie par FastAPI
Interface chatbot en HTML/CSS/JavaScript
Pipeline RAG avec LangChain
Recherche vectorielle avec Pinecone
Génération de réponse avec Groq LLM
Texte court à dire oralement :
La solution proposée est une application web complète. Le frontend permet à l’utilisateur de saisir une question. Le backend FastAPI reçoit cette question et la transmet au service RAG. Ce service recherche les passages pertinents dans Pinecone, puis utilise un modèle Groq pour générer la réponse finale. La réponse est ensuite affichée avec les sources.
Visuel recommandé :
Schéma : Utilisateur → Frontend → FastAPI → RAG → Pinecone/Groq
Design conseillé :
Schéma horizontal
Chaque composant dans un bloc
Éviter les détails de code
Temps estimé : 1 minute
